Beacon

Beacon acts as a visual or/and audible morse beacon. When called with an empty query string, this page will show this usage.

Beacon accepts the following query string parameters, of which only m (message) is required:

Parameter Meaning Default Remarks
m Message (a..z, 0..9) (none) A message is required and no default is provided
a Enable audio signal 0
v Enable video signal 1
t Dot (or "dit") length in milliseconds 50 Dash (or "dah") length is three dot lengths; gap between dots and dashes is one dot; gap between letters is three dots; gap between words is seven dots; see Wikipedia
f Audio signal frequency in hertz 700
c Video signal color as a (hex) RGB value ffffff

Examples

Silently morse the message "I AM THE WALRUS":

?m=i%20am%20the%20walrus

Use green signal color:

?m=i%20am%20the%20walrus&c=00ff00

Same message, this time also beep:

?m=i%20am%20the%20walrus&a=1

Lower audio pitch:

?m=i%20am%20the%20walrus&a=1&f=600
