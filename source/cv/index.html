Profile

I'm a 37 year old programmer/sysadmin/generalist from Kotka, Finland and have worked in the technology industry since 1997. Married since 2004, no children.

I've been "on the Internet" since the beginning of the 90's, before World Wide Web. I started working with Linux (1.2.13) in 1995 and set up my first public-facing web server in 1997. I also have some experience on other Unices. I wrote a lot of Perl code back in the day but have since semi-actively forgotten nearly everything about that. C, being such a simple language, still comes quite naturally after a little warmup. I write shell scripts (BaSH) as necessary, but Python has mostly replaced shell scripting as my general purpose tool of choice.

I've been a proponent of web standards before that became cool.

My current focus is on making things run fast and lean while keeping them simple and scalable. I'm trying to achieve these goals with the help of Python and Django on the server side and Backbone and related projects on the client side. Amazon Web Services have gradually started to play a bigger and bigger role in my way of working, and have simplified things tremendously.

The non-exhaustive near-term todo list: node.js; statistics.
